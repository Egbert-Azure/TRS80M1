# NEWDOS/80 variants

Per-build documentation of the modified NEWDOS/80 disks in this collection. Each
descends from NEWDOS/80 V2.0 (Apparat Inc.) but was extended by a different author.
Each build is documented from its own SYS overlays — author signatures, LIB command
tables, embedded strings — with release notes used as corroboration.
