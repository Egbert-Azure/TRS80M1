# Hardware

Documentation of the physical TRS-80 Model I hardware this collection was
built and used on: the modifications, boards, and build notes behind the
machine, as distinct from the disk images and software documented elsewhere.

Most of the collection's disks were created on the machine described here — an
Apparat/Radio Shack TRS-80 Model I fitted with an Expansion Interface, a
double-density floppy controller, a 256K bank-switch board, and a manual
speed-up. The banker documented below is the one the
[SideKick](../software/sidekick/README.md) multi-computer switch depends on.
