# G-DOS editions

G-DOS is the German NEWDOS — a NEWDOS/80 2.x derivative distributed by
**TCS Trommeschläger Computer GmbH** (St. Augustin) with the EACA Genie
computers, which are largely compatible with the TRS-80 Model I. G-DOS and its
modifications form their own lineage, documented here separately from the
[NEWDOS/80 variants](../NewDos/dos-versions/README.md).
